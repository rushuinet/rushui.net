# 介绍
go-zeus是一个使用golang技术栈开发的微服务框架（神王框架）
go-zeus是一个0依赖可插拔的微服务框架，基于微服务架构的抽象，其实现在zeus-plusin项目中，开发者可根据自己的技术栈进行灵活配置
